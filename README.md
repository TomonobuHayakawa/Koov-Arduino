# Koov-Arduino
KOOV Control on Arduino IDE

このライブラリは、Spresense拡張ボード向けのShieldボード（Spr-Koov）が前提になっています。
Spr-Koovはバージョン1/2があります。

Koov-Arduino.hの中の

```
/* Spr-Koov (Arduino Shield for Koov) version */
#define SPR_KOOV_VER	2
```
を変更してください。


このライブラリを使うにあたって、以下のライブラリのインストールが必要です。

https://github.com/TomonobuHayakawa/SoftPWM

https://github.com/lfhohmann/MMA8653
